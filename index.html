Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

Ensemble de pages et de ressources reliées entre elles, accessibles via Internet grâce au protocole HTTP/HTTPS.

Internet

Réseau mondial d’ordinateurs interconnectés permettant l’échange de données.

Cloud

Stockage et utilisation de données ou d’applications sur des serveurs distants accessibles en ligne.

W3C

Organisation internationale qui définit les standards du Web (HTML, CSS, etc.).

Partie 2 - Distinguez les langages du Web

Langages client

Langages exécutés dans le navigateur, comme HTML, CSS ou JavaScript, pour l’affichage et l’interactivité.

Navigateurs

Applications permettant de consulter les pages web (ex. Chrome, Firefox, Safari).

Langages serveur

Langages exécutés sur le serveur pour générer le contenu et gérer la logique (ex. PHP, Python, Node.js).

Frameworks

Ensembles d’outils et de bibliothèques facilitant le développement (ex. React, Django).

Bases de données

Systèmes organisant et stockant les données (ex. MySQL, MongoDB).

Site responsive

Site dont la mise en page s’adapte automatiquement à la taille de l’écran (ordinateur, tablette, mobile).

Application mobile

Logiciel conçu pour fonctionner sur un smartphone ou une tablette.

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

Ordinateurs ou services qui stockent et fournissent des ressources ou des données aux clients via Internet.

Adresse IP

Numéro unique identifiant un appareil connecté à un réseau.

Nom d’hôte et DNS

Le nom d’hôte est l’adresse lisible d’un site (ex. exemple.com). Le DNS traduit ce nom en adresse IP.

Protocoles

Ensemble de règles permettant la communication sur un réseau (ex. HTTP, FTP, TCP/IP).

Partie 4 - Identifiez les métiers du web

Développeurs

Professionnels qui conçoivent et programment les sites ou applications web.

Product Managers et chefs de projets

Responsables de la planification, de la stratégie et du suivi de la réalisation d’un produit numérique.

Designers

Créateurs de l’interface et de l’expérience utilisateur (UX/UI) d’un site ou d’une application.

Testeurs

Personnes chargées de vérifier la qualité, la performance et l’absence de bugs avant la mise en ligne.

Techniciens informatique

Experts qui assurent le support, la maintenance et la configuration des systèmes et réseaux.

Langages de développement web

HTML

Front-End

Squelette du site Web. Il définit la structure de base du contenu, comme les titres, les paragraphes, les images, etc.

CSS

Front-End

Décrie l'apparence visuelle d'une page web: Couleurs, polices, mises en pages, animations, etc... Il est présents pour completer HTML

JavaScript

Back-End

Language de programmation qui permet de rendre les pages web intéreactive: Formulaire, animations, jeux, etc...

PHP

Back-End

Language de script ocnçu pour le développement web. Il sert à générer des pages dynamiques et gérer des bases de données.

Java

Back-End

Language de programmation orienté objet, compilé en bytecode et exécuté par la machine virtuelle JVM. Utilisé pour les appli web, mobile, serveurs et logiciels d'entreprise.

Python

Back-End

Language de programmation polyvalent, lisible et concis. Utilisable pour le développement web, l'automatisation, IA, Science des données, etc...

C#

Back-End

Language de programmation orenté object développé par Microsoft. Il sert surtout pour les application Windows, développement de jeux (Unity), appli web et services Cloud.

SQL

Bases de données

Language standardisé de gestion de bases de données relationnelles. Permet de créer, lire, mettre à jour et supprimer des données ainsi que définir la structure des bases.

Boîte à outils du développeur

  • IDE comme VS Code : Un IDE (Integrated Development Environment) est un environnement de développement intégré : un logiciel qui regroupe éditeur de code, gestion de projet, débogueur, terminal, et parfois un compilateur.
  • Github : Plateforme en ligne de gestion de code source basée sur Git (un système de contrôle de version). Permet d'héberger des dépôts de code, de collaborer, automatiser des tests et partager des projets open sources.
  • Librairies / bibliothèques : Ensemble de fonctions, classes ou modules déjà écrits que l’on peut importer dans son code pour éviter de tout réinventer.
  • Frameworks : Structure ou « cadre » de développement qui impose une architecture, des conventions et des outils prêts à l’emploi, comme: Python, JavaScript, C#.
